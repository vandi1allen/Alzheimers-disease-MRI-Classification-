# Alzheimers-disease-MRI-Classification-

This project implements a CNN and a MobileNetV2 model for MRI-based Alzheimer’s classification.

Steps to run:
1. Open the notebook in Google Colab
2. Ensure dataset is placed in /Colab/AIProj/
3. Run all cells in order

Dependencies:
- TensorFlow
- NumPy
- Matplotlib
- scikit-learn
